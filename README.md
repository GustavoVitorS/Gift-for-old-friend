# Presente de aniversário - Alex

Site estático com tema tricolor, cards de fotos viráveis, mensagem em popup e botão de trilha sonora.

## Como publicar no GitHub Pages

1. Envie `index.html`, `README.md` e a pasta `assets` para o repositório.
2. Vá em **Settings > Pages**.
3. Em **Branch**, selecione `main` e `/root`.
4. Clique em **Save**.

Estrutura correta:

```txt
index.html
README.md
assets/
  audio/
    trilha-alex.mp3
  img/
    foto-01.jpg
    foto-02.jpg
    foto-03.jpg
    foto-04.jpg
    foto-05.jpg
    foto-06.jpg
    foto-07.jpg
```

Usuários permitidos no input: `Alex` ou `Calvin`.
