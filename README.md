# Birthday Tribute Website

This project is a personalized birthday tribute website created as a special gift for my friend.

The website was designed to celebrate friendship, memories, and meaningful moments through an interactive and emotional experience. It includes a custom login screen, animated visual effects, themed cards, a photo gallery reveal system, a birthday message popup, and a soundtrack button to make the experience more personal and memorable.

## Purpose

The main purpose of this project is to create a digital birthday gift that feels unique, personal, and interactive.

Instead of sending only a simple message or image, this website was built to deliver a complete experience, combining design, music, animation, photos, and a heartfelt message.

## Features

* Personalized birthday login screen
* Custom São Paulo inspired visual theme
* Interactive photo cards
* Click-to-reveal card animation
* Background animations
* Birthday message popup
* Soundtrack button with play/pause functionality
* Responsive layout for desktop and mobile devices
* Static website ready to be hosted on GitHub Pages

## Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## How to Use

Open the website and enter one of the allowed usernames to access the birthday tribute.

After entering, the visitor can interact with the photo cards, reveal memories, enable the soundtrack, and open the special birthday message.

## Deployment

This project can be deployed using GitHub Pages.

After uploading the files to a GitHub repository:

1. Go to the repository settings.
2. Open the Pages section.
3. Select the `main` branch.
4. Choose the `/root` folder.
5. Save and wait for GitHub Pages to generate the website link.
6. Deploy: https://gustavovitors.github.io/Gift-for-old-friend/
